1. FIRST_THOUSAND
   Name: "Baby Steps"
   Description: "Reach $1,000"
   How to Unlock:
     - Simply accumulate $1,000 in your balance
     - Should happen naturally in first 2-3 days
     - Win enough hands at blackjack
   Difficulty: TRIVIAL
   Missable: No
   Tips: You start with $50, need to win $950 more

2. FIRST_TEN_THOUSAND
   Name: "Getting Somewhere"
   Description: "Reach $10,000"
   How to Unlock:
     - Accumulate $10,000
     - Takes 5-10 days typically
     - Use basic blackjack strategy
   Difficulty: EASY
   Missable: No
   Tips: Proper bankroll management helps

3. FIFTY_THOUSAND
   Name: "Fifty Grand"
   Description: "Reach $50,000"
   How to Unlock:
     - Hit $50,000 balance
     - Requires consistent winning
     - About 15-20 days of play
   Difficulty: MEDIUM
   Missable: No
   Tips: Triggers "The Offer" event when combined with $250k milestone

4. HUNDRED_THOUSAND
   Name: "Six Figures"
   Description: "Reach $100,000"
   How to Unlock:
     - Accumulate $100,000
     - About 25-35 days typically
     - High stakes help
   Difficulty: MEDIUM
   Missable: No
   Tips: Game dynamics change at this level

5. QUARTER_MILLION
   Name: "Quarter Million"
   Description: "Reach $250,000"
   How to Unlock:
     - Hit $250,000 balance
     - Major milestone

   Difficulty: HARD
   Missable: No
   Tips: TRIGGERS "The Offer" event - mysterious deal offered
   Warning: This enables a major ending

6. HALF_MILLION

    Tips: Dangerous achievement, debt compounds

================================================================================

EARLY GAME PRIORITIES (Days 1-20):
✓ Focus on survival achievements (eating, sleeping, staying alive)
✓ Unlock basic companion achievements
✓ Complete first casino visit
✓ Meet all shop NPCs
✓ Collect easy money achievements ($100, $500, $1,000)

MID GAME PRIORITIES (Days 21-60):
✓ Relationship building achievements
✓ Item collection achievements
✓ Complete shop quests
✓ Blackjack skill achievements (10 wins, 50 wins, etc.)
✓ Companion collection achievements

LATE GAME PRIORITIES (Days 61-100):
✓ High-tier money achievements ($100,000, $500,000, $1,000,000)
✓ Ending-specific achievements
✓ Secret achievement hunting
✓ Rare event achievements
✓ Completion achievements

MULTIPLE PLAYTHROUGH ACHIEVEMENTS:
Some achievements require different paths:
• Good karma vs evil karma achievements
• Different ending achievements
• Contradicting choice achievements
Plan 3-5 playthroughs for 100% completion

MISSABLE ACHIEVEMENTS WARNING:
These can be permanently missed:
• One-time event achievements
• Day-specific achievements
• NPC quest achievements (if you anger them)
• Companion-specific achievements (if they die)
• Ending achievements (need right path)
